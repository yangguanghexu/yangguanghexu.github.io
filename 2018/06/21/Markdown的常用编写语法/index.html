我是第一次看到Markdown，之前一点都不了解。先说一下Markdown，Markdown 是 2004 年由 John Gruberis 设计和开发的纯文本格式的语法.通过简单的标记语法，它可以使普通文本内容具有一定的格式，在线论坛编写消息和快速创建富文本文档的文本编辑器都非常流行使用 Markdown 格式。

Markdown快速入门

这里贴一个网站，里面有很详细的语法和实例。
Markdown 语法说明

下面说一下比较常用的语法，很容易记，掌握后写文章没有一点问题。

标题的写法

H1 :# Header 1
H2 :## Header 2
H3 :### Header 3
H4 :#### Header 4
H5 :##### Header 5
H6 :###### Header 6

前面带#号，后面带文字，分别表示h1-h6，而且h1下面会有一条横线，注意#号后面有空格

无序列表可以用 * ， + ，- 号来创建，实际都是转换成了ul>li。

区块引用

区块引用的方法是在段落前面加一个 > 号，然后样式就是本篇文章第一段那个样子。

分割线

在一行中用三个以上的星号、减号、底线来建立一个分隔线，行内不能有其他东西。

超链接

[文字](链接地址)

链接的文字放在[ ]中，链接地址放在随后的( )中
